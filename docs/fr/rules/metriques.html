Les métriques sont accessibles à tous avec différents niveaux de granularité : vue détaillée pour l'Equipe produit concernée, agrégations pour les autres membres de l'organisation.

L'accès aux métriques n'implique pas l'accès aux données unitaires, celui-ci doit être contrôlé pour préserver la confidentialité.

Tous les environnements sont concernés.
