rules

[ ASYNC-FIRST ]

アプリケーション間の交換は非同期でなければなりません。

非同期交換は自然に弱いカップリング、分離、フロー制御（背圧）を可能にします。

同期通信は、アクションがそれを必要とするときのみ**考慮されるべきです。
